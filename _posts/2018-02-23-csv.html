About This Tool

This tool solves a couple of related problems:
1. Producing tables in markup languages like HTML and LaTeX is a pain.
2. Organizing information into tables in plain text documents is even more of a pain.

However, editing CSV files (in applications such as Excel, Numbers, etc.) is easy! So this tool takes CSV data as input and produces source code for tables in HTML and LaTeX, as well as a plain-text, monospaced table, suitable for use on practically any computer anywhere.
